Sie können Teilnehmer mittels einer CSV Datei importieren. Achten Sie darauf, dass die CSV Datei den untenstehenden Regeln entspricht.

  • Die Datei DARF NICHT grösser als 1MB sein.
  • Die Datei MUSS der Dateiendung ".csv" entsprechen.
  • Als Trennzeichen MUSS ein Komma vorhanden sein.
  • Die Klasse darf nicht länger als 6 Zeichen sein.
  • Das Geschlecht MUSS dem Zeichen "m" oder "w" entsprechen.
  • Das Geburtsdatum MUSS im Format DD.MM.YYYY sein. z.B. 05.08.2017
  • Die Spaltenüberschriften MÜSSEN vorhanden sein.
  • Die Spalten MÜSSEN in folgender Reihenfolge vorhanden sein:

    Klasse,Nachname,Vorname,Geschlecht,Adresse,PLZ,Ort,Geburtsdatum,Klassenlehrer
