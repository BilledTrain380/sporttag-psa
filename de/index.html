Wieso PSA

PSA ist ein Management System um Ihren Sport Event zu organisieren. Es ist eine standalone Program, welches nur Java benötigt. Falls Sie Java nicht benutzen möchten, können Sie auch das native App für macOS oder Windows verwenden.

Das Setup ist einfach. PSA benutzt automatisch den Application Support Ordner Ihres Systems. Keine Konfigurierung von Server Einstellungen nötig. Führen Sie einfach das Program aus und Sie können durchstarten.

Nutze PSA gratis

Kein Command Line Fan?

PSA kann mit einem minimalen Administrations GUI verwendet werden. Sie müssen dafür keinerlei Command Line Befehle benutzen.

  • Server status
  • Start Server
  • Stop Server

Das Administrations GUI ist ausschliesslich für macOS oder Windows native App verfügbar.
