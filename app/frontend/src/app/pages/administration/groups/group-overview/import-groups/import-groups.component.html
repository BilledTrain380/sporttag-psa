CSV Format

  • Must end with .csv extension
  • Must follow the attribute order:
    Group,Surname,Prename,Gender,Street,Zip Code,Town,Birthday,Group Coach
  • The gender must be a m for men and a w for woman
  • The birthday must follow the format dd.MM.YYYY ; e.g. 06.10.2009
  • The csv delimiter must be a comma ,
File is required Max file size is 1MiB Only .csv files are allowed
